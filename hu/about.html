Névjegy

Many people know from their own experience that it's not easy to install an Apache web server and it gets harder if you want to add MariaDB, PHP and Perl. The goal of XAMPP is to build an easy to install distribution for developers to get into the world of Apache. To make it convenient for developers, XAMPP is configured with all features turned on. In the case of commercial use please take a look at the product licenses, from the XAMPP point of view commercial use is also free. There are currently distributions for Windows, Linux, and OS X.

Alapítók

Kai 'Oswald' Seidler

Oswald volt az Apache Friends eredeti társalapítóinak egyike. 1999-ben szerzett diplomát a Berlini Műszaki Egyetemen Okleveles Informatikus fokozaton (egyenértékű a Számítástudományi mesteri címmel). A '90-es években ő hozta létre és kezelte Németország legnagyobb IRCnet kiszolgálóját, az irc.fu-berlin.de címen, és besegített a világ egyik legnagyobb névtelen FTP kiszolgálójának, az ftp.cs.tu-berlin.de kezelésében. 1993-tól 1998-ig tagja volt a nemzetközi „Projektgruppe Kulturraum Internet” csoportnak, a berlini székhelyű net kultúra és hálózat szervezési kutatási projektnek. 2006-ban Addison Wesley kiadásában jelent meg a harmadik könyve, a Das XAMPP-Handbuch. 2009-től 2011-ig Technológiai Evangélistaként szolgált a Web Tier termékekhez a Sun Microsystems/Oracle zászlaja alatt.

Kay Vogelgesang

Oswalddal együtt Kay alapította az Apache Friends projektet 2002-ben. Jelenleg szabadúszó rendszermérnökként dolgozik és számos könyvet írt az olyan webes technológiákról mint az Apache, MySQL és maga a XAMPP.
