Bitnami для XAMPP

Bitnami для XAMPP очень упрощает разработку приложений на PHP поверх ApacheFriends XAMPP, путем установки WordPress и многих других приложений с открытым исходным кодом поверх XAMPP.

Системные требования

Модули Bitnami XAMPP требуют XAMPP или Bitnami LAMP, MAMP или WAMP стеки уже установленными. Они могут быть использованны с помощью XAMPP (установщик или zip-архив) версии 1.7.4 или позднее на Windows, версии 1.8.1 или позднее на Linux, и версии 1.8.2 или позднее на Mac OS X.

Bitnami для XAMPP работает корректно поверх портативной версии. Из-за требований приложения, после установки приложения вы не можете переместить или переименовать свою папку с XAMPP.

Установка

Модули Bitnami XAMPP требуют XAMPP или Bitnami LAMP, MAMP или WAMP стеки уже установленными. Они могут быть использованны с помощью XAMPP (установщик или zip-архив) версии 1.7.4 или позднее на Windows, версии 1.8.1 или позднее на Linux, и версии 1.8.2 или позднее на Mac OS X.

Перед началом установки модуля убедитесь, что сервер XAMPP MySQL запущен.

Во время процесса установки, вам будет предложено указать папку установки XAMPP. Каталог установки по умолчанию обычно C:\xampp на Windows, /opt/lampp на Linux и /Applications/XAMPP на Mac OS X.

Модули Bitnami для XAMPP требуют root пароля MySQL для создания баз данных приложения. Пароль XAMPP по умолчанию для учетной записи MySQL root — пустая строка. Если вы изменили этот пароль, вам будет необходимо ввести его в процессе установки модуля.

Расположение файлов

Приложения Bitnami будут установлены в папке XAMPP, в подкаталоге apps/. Все приложения настроены со своими файлами в следующих местах:

  • INSTALL-DIR/apps/APP-NAME/htdocs: Файлы приложений
  • INSTALL-DIR/apache/conf/httpd.conf: Файлы конфигурации Apache
  • INSTALL-DIR/apps/APP-NAME/conf/httpd-prefix.conf (в предыдущих версиях, wordpress.conf): Файл конфигурации Apache для приложения

Чтобы получить доступ к приложению, перейдите на http://localhost/НАЗВАНИЕ-ПРИЛОЖЕНИЯ. К примеру, WordPress будет доступен на веб-странице http://localhost/wordpress. Вы также можете получить доступ к приложениям с главной страницы XAMPP под вкладкой "Applications" или "Приложения".

Реквизиты для входа

Реквизиты администратора для каждого приложения могут быть настроены из установщик. Стандартное имя пользователя - "user", а пароль, какой вы ставили при установке.
