XAMPP хостинг

XAMPP обеспечивает идеальную локальную среду разработки, но не предназначена для производственных развертываний. Мы хотим максимально упростить размещение PHP-приложений, созданных с помощью XAMPP. Мы начинаем с предоставления подробных руководств о том, как запустить XAMPP в облаке с помощью Microsoft Azure, Google Cloud Platform и Amazon Web Services. Вы также можете перенести свое приложение WordPress с XAMPP на высокодоступное, готовое к работе развертывание WordPress на Amazon Web Services.

Пожалуйста, свяжитесь с нами по эл. почте hello (собака) apachefriends.org, если у вас есть отзывы о руководствах или вы хотели бы внести свой вклад в перевод.
